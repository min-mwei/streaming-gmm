class GradientAggregator extends Serializable

Distributed aggregator of relevant statistics

In each worker it computes and aggregates the current batch log-likelihood, the regularization values for the current parameters and the gaussianGradients for each data point. The class structure is based heavily on Spark's ExpectationSum
