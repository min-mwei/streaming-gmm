class ConjugatePrior extends Regularizer

Conjugate prior regularization for all the mixture's parameters; this means an Inverse-Wishart prior over the covariance matrices, a Normal prior over the means and a Dirichlet distribution prior over the weights.

  25. var regularizingMatrix: DenseMatrix[Double]

    Parameter block matrix [A B; C D].

    Parameter block matrix [A B; C D]. The blocks are:

    A = iwMean + kappa * normalMean * normalMean.t

    B = kappa * normalMean

    C = kappa * normalMean.t

    D = kappa

    kappa = degFrredom + dim + 2
